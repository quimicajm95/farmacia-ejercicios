Conceptos Fundamentales de Termoquímica

Entalpía (ΔH)

Es el contenido de calor de un sistema a presión constante. La variación de entalpía (ΔH) indica si una reacción es exotérmica (ΔH < 0) o endotérmica (ΔH > 0).

ΔH°reacción = ΣΔH°f(productos) - ΣΔH°f(reactivos)

Entropía (S)

Es una medida del desorden o aleatoriedad de un sistema. La variación de entropía (ΔS) indica si aumenta (ΔS > 0) o disminuye (ΔS < 0) el desorden.

ΔS°reacción = ΣS°(productos) - ΣS°(reactivos)

Energía Libre de Gibbs (ΔG)

Indica la espontaneidad de una reacción a temperatura y presión constantes. Si ΔG < 0, la reacción es espontánea.

ΔG = ΔH - TΔS

Relación entre ΔG, ΔH y ΔS

La espontaneidad de una reacción depende de la combinación de ΔH y ΔS:

ΔH negativo y ΔS positivo → Siempre espontánea (ΔG < 0)
ΔH positivo y ΔS negativo → Nunca espontánea (ΔG > 0)
ΔH negativo y ΔS negativo → Espontánea a bajas temperaturas
ΔH positivo y ΔS positivo → Espontánea a altas temperaturas

Pasos para Resolver Problemas Termoquímicos

1. Escribir y balancear la ecuación química.
2. Calcular ΔH°reacción usando entalpías de formación.
3. Calcular ΔS°reacción usando entropías estándar.
4. Calcular ΔG°reacción usando ΔG = ΔH - TΔS.
5. Determinar la espontaneidad de la reacción.

Practica tus conocimientos con estos ejercicios interactivos

1 Disolución de Cloruro de Zinc

A partir de los datos siguientes:

Sustancia ΔH°f (kJ/mol) S° (J/(K·mol))
ZnCl₂(s) -416,1 108,5
Zn²⁺(ac) -152,5 -106,6
Cl⁻(ac) -167,5 55,2

Calcule la energía libre molar a 25 °C del proceso representado por:

ZnCl₂(s) + agua → Zn²⁺(ac) + 2Cl⁻(ac)

a) Calcule ΔH° para la reacción (en kJ/mol)

b) Calcule ΔS° para la reacción (en J/(K·mol))

c) Calcule ΔG° para la reacción a 25°C (en kJ/mol)
